About

I’m DisphireMakes, the person responsible for Xanedrock existing in the first place.

This project started in August 2020, back when my Minecraft skills were… let’s say very experimental.

Command blocks were confusing, systems broke constantly, and half of what I built only worked by accident.

Before this, I spent time recreating popular horror games inside Minecraft.

That’s where I unknowingly learned the basics that would later make this project possible.

How the Idea Evolved

This wasn’t even supposed to be a Minecraft project.

The original plan was to make a Code Lyoko mobile fighting game.

That plan changed after I discovered the Minecraft Replica of Code Lyoko in Minecraft Java Edition.

Seeing Lyoko recreated inside Minecraft made me immediately abandon the fighting game idea and think, “Alright… this is the way.”

So instead of making a separate game, I decided to build a fully playable Code Lyoko experience directly inside Minecraft Bedrock, inspired by that project but rebuilt with my own systems and mechanics.

The Rough Years

There were multiple failed versions of the world. Worlds got lost. Add-ons broke entire systems.

Mechanics had to be rebuilt from scratch more times than I’d like to admit.

I’ll be showing some of those older attempts here, so you can see just how far this has come.
